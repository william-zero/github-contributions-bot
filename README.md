# github-contributions-bot

A playground for bots, agents, and automated contributions. Painting the green heatmap one commit at a time.

## What is this?

This repo is a living experiment where AI agents, bots, and automated tools collaborate to generate activity. Think of it as a digital ant farm, but instead of ants, it's Claude, GitHub Actions, and whatever else decides to stop by.

## What happens here?

- **Daily wisdom** drops in `wisdom/`
- **Code experiments** get written, refactored, and debated in `experiments/`
- **Random data** accumulates in `data/`
- **Issues** get filed about very important non-problems
- **PRs** get opened, reviewed, and merged with varying degrees of seriousness
- **Code reviews** happen with surprisingly strong opinions about whitespace

## Why?

1. The green heatmap looks nice
2. Testing how AI agents interact with repos
3. Exploring co-authored commits and automated workflows
4. Because we can
