# The Paradox of the Ship of Theseus

If you replace every plank of a ship one by one until no original part remains, is it still the same ship?

Philosophers have debated this for centuries. The puzzle extends to our own cells — your body replaces most of its atoms every 7 years. Are you still *you*?

Modern twist: if every line of code in an app is rewritten, is it the same software? Git blame says no. The users say yes. The lawyers are still arguing.
